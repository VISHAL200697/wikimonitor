Variables

These variables represent properties of the recent change event. You can use them in your filter expressions.

Variable Type Description
user String Username or IP of the editor.
title String Title of the page.
comment String Edit summary/comment.
wiki String Database name of the wiki (e.g., 'enwiki').
type String Type of change ('edit', 'new', 'log', etc.).
namespace Integer Namespace ID of the page.
bot Boolean True if the edit was made by a bot.
minor Boolean True if marked as a minor edit.
added_lines String Content of lines added in this edit.
removed_lines String Content of lines removed in this edit.
user_groups List<String> List of user groups (e.g., 'sysop', 'bot').
user_rights List<String> List of user rights.
id Long Event ID.
page_id Long ID of the page.
title_url String URL of the page title.
parsedcomment String Parsed edit summary (HTML).
timestamp Long Timestamp of the edit.
patrolled Boolean True if the edit was patrolled.
notify_url String URL for notifications.
server_url String Base URL of the server.
server_name String Hostname of the server.
server_script_path String Path to the server script.
flagged Boolean Internal flag (true if flagged by filter).
abuseType String Internal abuse type classification.
user_name String Alias for 'user'.
page_namespace Integer Alias for 'namespace'.
old_size Long Size of the page before the edit.

String Functions

contains(text, value)

Checks if text contains value (case-sensitive).

contains(title, 'Java')

containsIgnoreCase(text, value)

Checks if text contains value (case-insensitive).

containsIgnoreCase(comment, 'vandalism')

length(text)

Returns the length of the string.

length(comment) > 50

normalizeArabic(text)

Normalizes Arabic text (removes diacritics, unifies alef forms, etc.).

contains(normalizeArabic(title), 'محمد')

startsWith(text, value)

Checks if text starts with value.

startsWith(title, 'User:')

endsWith(text, value)

Checks if text ends with value.

endsWith(title, '.js')

equals(text, value)

Checks if text equals value (case-sensitive).

equals(user, 'Admin')

equalsIgnoreCase(text, value)

Checks if text equals value (case-insensitive).

equalsIgnoreCase(user, 'admin')

isEmpty(text)

Checks if text is null or empty.

isEmpty(comment)

isBlank(text)

Checks if text is null, empty, or whitespace.

isBlank(comment)

count(text, needle)

Returns the number of occurrences of needle in text.

count(added_lines, 'link') > 5

lower(text)

Converts text to lowercase.

lower(title)

upper(text)

Converts text to uppercase.

upper(title)

trim(text)

Removes leading and trailing whitespace from text.

trim(comment)

removeWhitespace(text)

Removes all whitespace from text.

removeWhitespace(title)

Regex Functions

matches(text, regex)

Returns true if text matches the regular expression.

matches(title, '^User:.*')

rcount(text, regex)

Returns the number of matches of the regex in the text.

rcount(added_lines, 'https?://') > 5

List Functions

in(value, list)

Returns true if value is present in the list.

in('sysop', user_groups)

anyContains(text, list)

Returns true if text contains any string from list.

anyContains(comment, {'bad', 'word'})

allContains(text, list)

Returns true if text contains all strings from list.

allContains(comment, {'required', 'terms'})

Time Functions

hour(time)

Returns the hour of the day (0-23).

hour(now())

dayOfWeek(time)

Returns the day of the week.

dayOfWeek(now()) == 'MONDAY'

isNightTime(time, start, end)

Returns true if the time is between start (inclusive) and end (exclusive) hours.

isNightTime(now(), 22, 6)

Examples

Detect potential vandalism (large removal of content):

old_size > 1000 && (old_size - length(added_lines) + length(removed_lines)) > 500

Monitor specific user on specific wiki:

wiki == 'enwiki' && user == 'TargetUser'

Filter for main namespace edits by anonymous users:

namespace == 0 && matches(user, '^\d+\.\d+\.\d+\.\d+$')
